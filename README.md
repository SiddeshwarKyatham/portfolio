# Siddeshwar · Folio

Personal portfolio of **Kyatham Sai Siddeshwar** – Frontend & Full‑Stack Developer (MERN focus).

## Tech stack

- Bun
- Vite
- React + TypeScript
- Tailwind CSS + shadcn/ui
- Framer Motion

## Running locally

```bash
bun install
bun run dev
```

Then open `http://localhost:8080` in your browser.
